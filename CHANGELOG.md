# Change Log

## [0.1.0] - 2025-11-12

### Added
- Syntax highlighting for comments starting with semicolon (`;`)
- Syntax highlighting for strings within double quotation marks (`"`) with escape character recognition
- Syntax highlighting for numbers in various bases (binary, decimal, hexadecimal) and single-quoted characters
- Syntax highlighting for 8051/8052 assembly language mnemonics
- Syntax highlighting for common built-in registers (`A`, `ACC`, `DPTR`, `R0`~`R7`, etc.)
- Syntax highlighting for common assembly directives (`.org`, `.db`, `.equ`, `.if`, `macro`, etc.)
- Syntax highlighting for SDCC-specific assembly directives (`.module`, `.area`, `.globl`, etc.)
- Syntax highlighting for Keil-specific assembly directives

## [0.0.1] - 2025-11-11

- Create a new empty extension and publish to the marketplace
